# 特点
- 预设多档频率。
- 在默认频率算力下比蚂蚁官方固件节省150W功率，按0.3元/度电计算，每台机器1天可以省1元电费。
- 最高可超频至700算力（相应功耗会增加）。
- 四块算力板可以设置不同的频率/电压。
- 解锁全频率/电压，用户可以自行调节适合算力板体质的最优效果
